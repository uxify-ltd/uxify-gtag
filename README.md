# UXify Tag Template

This template can be used to easily load the Uxify script in your website via Google Tag Manager.

A Uxify subscription is required. To get one, please visit [The Uxify Website](https://uxify.com).

## Instructions

1. Go to your [Google Tag Manager Console](https://tagmanager.google.com)
2. Create a new tag under the Tags section.
3. For Tag Configuration, choose the "Community Template Gallery" and find this template called "Uxify"
4. Enter your Uxify apikey.
5. Under "Advanced Settings" check the option "Only fire this tag in published containers."
6. Under "Advanced Settings" for "Tag firing options" choose the option "Once pre page".

Create your tag and add it to your website. Congratulations, you now have Uxify :)
